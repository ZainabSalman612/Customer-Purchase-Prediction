# Customer Purchase Prediction

A machine learning project to predict whether a customer will purchase a product based on their behavior on the website.

## Features
- Time on site
- Pages viewed
- Device type
- Referral source

## Skills Learned
- Data preprocessing (encoding categorical variables, scaling)
- Logistic Regression
- Random Forest
- Model evaluation (accuracy, precision)

## Project Structure
- `notebooks/`: Jupyter notebooks for exploration and modeling
- `src/`: Source code (if any)
- `data/`: Data files
- `requirements.txt`: Python dependencies

## Setup
1. Create a virtual environment: `python -m venv .venv`
2. Activate: `.venv\Scripts\activate` (Windows)
3. Install dependencies: `pip install -r requirements.txt`
4. Run the notebook: `jupyter notebook notebooks/customer_purchase_prediction.ipynb`
